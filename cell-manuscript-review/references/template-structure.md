# STAR Methods template — section structure

This is the required section order for a Cell manuscript using the STAR Methods template (April 2026 revision). Use this as a checklist for Step 1 of the workflow to verify the manuscript's section order matches the template.

## Section order

Sections should appear in this exact order. Required sections must be present; optional sections are present only when applicable.

| Order | Section | Status | Notes |
|---|---|---|---|
| 1 | Title | required | ≤145 characters |
| 2 | Author list with footnotes | required | Includes corresponding author marks (*), lead contact indicator |
| 3 | Affiliations | required | Numbered, complete (department, city, state, postal code, country) |
| 4 | Corresponding author emails | required | One line per corresponding author |
| 5 | MOTIVATION | journal-specific | Only for *Cell Reports Methods*. Delete for Cell. |
| 6 | SIGNIFICANCE | journal-specific | Only for *Cell Chemical Biology*. Delete for Cell. |
| 7 | Abstract | journal-specific | Only for *Med* (semi-structured). For Cell, use Summary instead. |
| 8 | SUMMARY | required | Single paragraph ≤150 words, no citations |
| 9 | KEYWORDS | required | Up to 10, comma-separated |
| 10 | INTRODUCTION | required | No subheadings, ends with brief achievement statement |
| 11 | DESIGN | article-type-specific | Only for Technology articles. Delete otherwise. |
| 12 | RESULTS | required | Subheadings should be informative claims, not topic labels |
| 13 | DISCUSSION | required | Must end with Limitations of the Study subsection |
| 14 | RESOURCE AVAILABILITY | required | Three subsections: Lead Contact, Materials Availability, Data and Code Availability |
| 15 | ACKNOWLEDGMENTS | required | Funding sources with grant numbers |
| 16 | AUTHOR CONTRIBUTIONS | required | CRediT format preferred |
| 17 | DECLARATION OF INTERESTS | required | Genuine COI disclosure |
| 18 | DECLARATION OF GENERATIVE AI | required if AI used | Tool name + purpose + responsibility statement |
| 19 | SUPPLEMENTAL INFORMATION | required if SI exists | Index of Document S1 and separate files |
| 20 | FIGURE TITLES AND LEGENDS | required | Each figure: title + legend |
| 21 | TABLES AND TEXT BOXES | required if tables exist | Main-text tables only |
| 22 | REFERENCES | required | Numbered, full bibliographic info, DOI required |
| 23 | STAR★METHODS | required | Container heading for the next five subsections |
| 23a | KEY RESOURCES TABLE | required | Template category headers only |
| 23b | EXPERIMENTAL MODEL AND STUDY PARTICIPANT DETAILS | required | Computational studies state this explicitly |
| 23c | METHOD DETAILS | required | Sufficient for reproduction |
| 23d | QUANTIFICATION AND STATISTICAL ANALYSIS | required | Tests, software, n values |
| 23e | ADDITIONAL RESOURCES | required | Public asset URLs in "Description: URL" format |

## Common deviations to flag

These deviations from the template appear frequently in drafts and should be flagged in the review:

**Missing Additional Resources subsection** — easy to miss because the last natural-feeling subsection is Quantification and Statistical Analysis. Authors often stop there. The Additional Resources subsection is required and is the appropriate place for leaderboard URLs, repository links, and clinical trial registry numbers.

**Author Contributions and Declaration of Interests swapped** — happens when authors copy-paste their CRediT statement into the wrong section, then never write the actual COI disclosure. The fingerprint is: Declaration of Interests heading followed by CRediT-style text ("Conceptualization, X.Y.; methodology, A.B."), and Author Contributions heading followed by either nothing or template-fictional initials.

**Template placeholder text retained** — the Cell template uses fictional authors (Rosalind Franklin, Lin Lanying, Katherine Johnson) with initials S.C.P., S.Y.W., A.B., M.E., A.N.V., N.A.V., M.E.V., C.K.B., A.A.D., N.L.W., A.A., and so on. If any of these initials appear in the manuscript's Author Contributions, the author copy-pasted the template's example and forgot to replace it. The actual authors' initials should appear.

**Missing Limitations of the Study** — required for Cell but often missing because authors don't feel they have "limitations" worth highlighting. Required regardless.

**Data and Code Availability written as prose** — template requires the three-bullet structure (Data, Code, Additional information). Authors who haven't read the template carefully write it as a single paragraph.

**Highlights and eTOC blurb inside the main manuscript** — these belong in a separate Word file. Authors who use the STAR Methods template literally as a single-file Word document sometimes include them at the top, which is wrong.

## Optional and conditional sections

- **MOTIVATION**: only for *Cell Reports Methods*. Delete for Cell.
- **SIGNIFICANCE**: only for *Cell Chemical Biology*. Delete for Cell.
- **Abstract** (structured): only for *Med*. Cell uses Summary instead.
- **DESIGN**: only for Technology articles. Delete otherwise.
- **Limitations (subsection of Discussion)**: heading is "Limitations" not "Limitations of the Study" for Technology articles only. For Cell research articles, use "Limitations of the Study".

## Section heading capitalisation in the template

The template uses ALL CAPS for top-level section headings (SUMMARY, INTRODUCTION, RESULTS, DISCUSSION, etc.). Some authors convert these to sentence case in their final draft — both styles are acceptable to Cell, but consistency matters. If the manuscript uses mixed case (some headings ALL CAPS, others Title Case, others sentence case), flag as a MINOR issue.
