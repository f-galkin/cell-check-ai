# Cell Final File Requirements — Checklist

Distilled from the CELLFFC PDF and the STAR Methods article template (April 2026 revision), this document is the working checklist for Step 2 of the review workflow. Work through it section by section while reading the manuscript.

Each item is tagged with the severity to assign if it fails the check.

## Title page

- [ ] **Title is ≤145 characters including spaces.** (CRITICAL if over.) Use sentence- or title-case consistently throughout the title — mixing the two within one title is a MAJOR issue.
- [ ] **Title avoids jargon and uncommon abbreviations.** MINOR if specialised acronyms appear without spelling out.
- [ ] **Author list uses full names, not initials.** First name + Last name format. MAJOR if initials-only.
- [ ] **Author surnames have no space before the superscript footnote indicator.** "Jane Doe1,2,*" is correct; "Jane Doe 1,2,*" is wrong. MINOR.
- [ ] **Each affiliation includes department/subunit, institution name, city, state/region, postal code, and country.** MAJOR if postal code or department missing. Commercial entities without departments may use institution name + postal code + country, but city and state still required.
- [ ] **Each corresponding author has an email address.** CRITICAL if missing.
- [ ] **Lead contact is signalled with a dedicated numeric footnote in the author byline.** Example: "Jane Doe1,3,*" where footnote 3 reads "Lead contact". MAJOR if absent — having only a "Correspondence (lead contact):" line at the bottom does not replace the byline footnote.
- [ ] **Senior author footnote and "These authors contributed equally" footnotes** are appropriate when relevant. INFO if relevant but missing; not always applicable.
- [ ] **There must be exactly one Lead Contact**, who must also be a corresponding author. CRITICAL if multiple Lead Contacts named.

## Summary

- [ ] **Single paragraph, ≤150 words.** CRITICAL if over.
- [ ] **No reference citations.** CRITICAL if citations present.
- [ ] **No non-standard abbreviations.** MAJOR if uncommon acronyms appear without definition.
- [ ] **Covers: background/purpose, principal results and approach, broader significance.** MINOR if any of the three is missing.

## Keywords

- [ ] **Up to 10 keywords.** CRITICAL if over.
- [ ] **Comma-separated, with abbreviations as separate keywords from their spelled-out form.** E.g., "embryonic stem cell, ESC". MINOR.

## Introduction

- [ ] **No subheadings.** MAJOR if any.
- [ ] **Ends with a brief statement of what has been achieved.** MINOR if absent.

## Results

- [ ] **Subheadings are informative claims, not topic labels.** "Factor X requires factor Y to function in process Z" not "Analysis of factors X and Y". MINOR.
- [ ] **Subheading capitalisation consistent.** Either all sentence case or all title case. MINOR if mixed.
- [ ] **Acronyms defined at first appearance.** MINOR.
- [ ] **IUPAC/IUBMB nomenclature for chemicals/biomolecules.** MAJOR if non-standard names used.
- [ ] **Organism names: genus + species spelled out in full at first appearance, italicised.** *Drosophila melanogaster* on first mention, *D. melanogaster* afterward. MAJOR if not italicised; MINOR if abbreviated on first use.
- [ ] **Gene symbols italicised; protein products not italicised.** MAJOR if reversed or inconsistent.
- [ ] **No footnotes (except in author list and tables).** MAJOR if any.
- [ ] **Section headings and subheadings not numbered.** MAJOR if numbered.
- [ ] **The words "new" and "novel" are not used as priority claims** or to refer to chemical compounds or structures. MAJOR if used as priority claims; INFO if used in benign context (e.g., "develop new tools for X").
- [ ] **Period for decimals, not comma.** "50.25" not "50,25". MINOR.
- [ ] **Citations not used as parts of speech.** Write "see Smith et al.1" not "see 1". MINOR.
- [ ] **All figures cited in numerical order in the main text.** CRITICAL if Figure 3 cited before Figure 2; MAJOR if a figure is not cited at all.
- [ ] **All supplemental figures and tables cited in the main text at least once.** MAJOR if cited in legend only or not at all.

## Discussion

- [ ] **Limitations of the Study subsection present.** MAJOR if absent.
- [ ] **Does not simply repeat results.** MINOR if it does.
- [ ] **Places findings in broader context.** MINOR if absent.

## Resource Availability

- [ ] **Heading is "Resource Availability"** with three required subheadings: Lead Contact, Materials Availability, Data and Code Availability. CRITICAL if any subheading missing.
- [ ] **Lead Contact subsection** gives full name and email of the Lead Contact, matching the byline footnote. MAJOR if mismatch.
- [ ] **Materials Availability subsection** is present even if no new materials were generated (in which case it states so). MAJOR if absent.
- [ ] **Data and Code Availability subsection** uses the three-bullet structure: (1) Data, (2) Code, (3) Additional information. MAJOR if written as a single paragraph instead of bullets.
- [ ] **Each bullet uses approved phrasing** from the template (e.g., "deposited at [repository] and publicly available as of the date of publication"). MINOR if non-standard phrasing.

## Acknowledgments

- [ ] **Includes funding sources with grant numbers.** MAJOR if funding undisclosed.
- [ ] **Acknowledges non-author contributions** where appropriate. MINOR if missing for contributions mentioned elsewhere.

## Author Contributions

- [ ] **Uses CRediT taxonomy format** ("Conceptualization, A.B.; Methodology, C.D.; …"). MINOR if traditional prose format used.
- [ ] **Initials match the author list.** CRITICAL if placeholder initials from the template remain: S.C.P., S.Y.W., A.B., M.E., A.N.V., N.A.V., M.E.V., C.K.B., A.A.D., N.L.W. are the template's fictional authors and must NOT appear unless they are coincidentally real authors.
- [ ] **Disambiguation when multiple authors share a first initial.** E.g., A.Z., A.Al., A.Am. for three Alex's. MINOR if ambiguous.
- [ ] **All authors appear in at least one CRediT role.** MAJOR if an author is missing from the contributions block.
- [ ] **All credited initials map to a real author.** CRITICAL if an initial appears that does not correspond to any author (often indicates template residue).

## Declaration of Interests

- [ ] **Present as a separate section** after Author Contributions. CRITICAL if absent.
- [ ] **Contains genuine COI disclosure**, not CRediT or other content. CRITICAL if it contains CRediT-style text (likely swapped with Author Contributions).
- [ ] **Discloses employment, equity, patents, consulting, advisory roles for all authors with potentially competing interests.** Use the form "X is an employee and shareholder of COMPANY" or "X is a founder of COMPANY and a member of its scientific advisory board". MAJOR if commercial affiliations are mentioned in author list but not disclosed here.
- [ ] **Ends with "The authors declare no other competing interests"** or similar wrap-up if some authors have nothing to disclose. MINOR if missing.
- [ ] **Matches the separate Declaration of Interests PDF form** that will be uploaded. INFO — verify the user will reconcile the two.

## Declaration of Generative AI

- [ ] **Present if AI tools were used in the writing process.** MAJOR if AI was used (evidenced by phrasing patterns or user statement) but no declaration.
- [ ] **Names the tool, the purpose, and includes the responsibility statement** ("the author(s) reviewed and edited the content as needed and take(s) full responsibility for the content of the publication"). MINOR if any element missing.

## Supplemental Information

- [ ] **Lists Document S1 first, with contents** (e.g., "Document S1. Figures S1–S6, Table S1"). MAJOR if Document S1 not listed.
- [ ] **Lists separately each non-PDF supplemental file** (Excel tables, videos, large datasets). MAJOR if a separate Excel table is referenced in text but not listed.
- [ ] **Does NOT include titles or legends of supplemental figures** — these belong in the supplemental PDF. MINOR if titles given here.
- [ ] **Every supplemental figure and table cited in main text is accounted for in the SI listing.** CRITICAL if Figure S4 is cited in text but Document S1 listing says "Figures S1–S3" only.

## Figure titles and legends

- [ ] **Figure titles use sentence case** and are concise (re-using Results subheading where possible). MINOR if title-case.
- [ ] **Legends define error bars, scale bars, statistical tests, p-value thresholds.** MAJOR if statistics undefined.
- [ ] **Legends state n (biological and technical replicates).** MAJOR if absent.
- [ ] **Legends do not include panel labels in the title.** MINOR if "(A) Foo, (B) Bar" appears in title instead of legend body.

## Tables

- [ ] **Created with Word's Table tool** (not tabs/spaces or Excel embedded). MAJOR if non-table format.
- [ ] **Each cell has discrete data; "N/A" or en-dash for missing.** MINOR.
- [ ] **Black and white, no coloured shading.** MAJOR if shaded.
- [ ] **Numbered Table 1, Table 2 (not Table 1A, 1B).** MINOR.
- [ ] **All tables cited in main text.** MAJOR if uncited.
- [ ] **Footnotes use superscript lowercase letters (a, b, c)**, not numbers or symbols. MINOR.

## References

- [ ] **Every in-text citation has a corresponding entry in the reference list.** CRITICAL if not.
- [ ] **Every entry in the reference list is cited at least once in text.** MINOR if entry uncited.
- [ ] **Format: LastName, F. (year). Title. Journal abbreviation volume, pages. DOI.** MAJOR if inconsistent (e.g., full first names in one entry, initials in others).
- [ ] **Preprints formatted as "Preprint at arXiv, https://doi.org/…" or "Preprint at bioRxiv, https://doi.org/…".** MINOR if arXiv.org or other variants used.
- [ ] **No duplicate DOI URLs.** MAJOR if any entry has the DOI repeated (common export artefact from reference managers).
- [ ] **Every entry has authors, year, title, venue, identifier (DOI or URL).** CRITICAL if any of these is missing.
- [ ] **Volume numbers italicised.** MINOR.

## STAR Methods

The STAR Methods section appears after References and contains the following subsections in order:

- [ ] **Key Resources Table** — present, uses template category headers (Antibodies, Bacterial and virus strains, Biological samples, Chemicals/peptides/recombinant proteins, Critical commercial assays, Deposited data, Experimental models: Cell lines, Experimental models: Organisms/strains, Oligonucleotides, Recombinant DNA, Software and algorithms, Other), no custom headers. MAJOR if custom headers used.
- [ ] **KRT references use "Author et al.N" form**, not bare reference number. MAJOR if a SOURCE column entry is just "50".
- [ ] **Experimental Model and Study Participant Details** — present. Computational-only studies state this explicitly. MAJOR if missing.
- [ ] **Method Details** — present with sufficient detail to reproduce. MAJOR if missing.
- [ ] **Quantification and Statistical Analysis** — present with statistical tests and software listed. MAJOR if missing.
- [ ] **Additional Resources** — present as the final subsection, listing public asset URLs and clinical trial registry numbers in "Description: URL" format. MAJOR if missing (common omission).

## Formatting consistency

- [ ] **No body paragraphs styled as headings.** CRITICAL when a sentence carries Heading 1 style — it will render as oversized blue text.
- [ ] **Thousands separators applied consistently.** Either all 4+ digit integers use commas (1,000; 32,768) or all use abbreviations (1K; 32K) — but not mixed within the same paragraph. MINOR.
- [ ] **Product/system names spelled consistently throughout.** MINOR if "LongevityBench" and "Longevity Bench" both appear.
- [ ] **Figure callouts use consistent spacing** ("Figure 5", not "Figure5") and consistent boldness. MINOR.

## Items to prepare separately (INFO-level reminders)

These are not in the main manuscript file but are part of the submission package. Flag each one if absent or non-compliant:

- [ ] **Highlights and eTOC blurb in a single separate Word file.** Highlights: 3–4 bullets, ≤85 characters each, no abbreviations. eTOC blurb: ~50–80 words, third person, "First Author et al." style.
- [ ] **Graphical abstract.** Mandatory for Cell. Exactly 1200 × 1200 pixels, TIFF/PDF/JPG, no surrounding white space, self-explanatory without legend. BioRender or Adobe Illustrator preferred — not PowerPoint.
- [ ] **Declaration of Interests PDF form.** Cell Press download, completed electronically, content matches the in-manuscript Declaration of Interests statement.
- [ ] **High-resolution main figure files.** Individual TIFFs at ≥300 DPI for photographs, ≥600 DPI for line art. All panels of one figure on a single page.
- [ ] **Supplemental files.** Document S1 (PDF), separate Excel tables (if any), supplementary videos (if any), Supplementary Files referenced by name.
